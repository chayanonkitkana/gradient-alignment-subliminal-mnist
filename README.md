# gradient-alignment-subliminal-mnist

This repository provides code to reproduce the results and figures from the paper:

**Sustained Gradient Alignment Mediates Subliminal Learning in a Multi-Step Setting: Evidence from MNIST Auxiliary Logit Distillation Experiment**

OpenReview Forum: https://openreview.net/forum?id=UJM4H9oLJN

## Repository overview

### Figure 1
Figure 1 is produced using:

- `subliminal-learning.ipynb`  
  (formerly `e3-crt-period.ipynb`)
- `subliminal-learning-per-step-statistics.ipynb`  
  (formerly `e3-plots.ipynb`)

### Figures 2A and 2B
Figures 2A and 2B are produced using:

- `conflicting-gradients.ipynb`  
  (formerly `e2-conflicting-gradients.ipynb`) for Figure 2A
- `liminal-anchor.ipynb`  
  (formerly `e4-liminal-anchor.ipynb`) for Figure 2B
- `ctrl-vs-int-multi-seed-plot.ipynb`  
  (formerly `e4-plots.ipynb`) for plotting results across seeds
